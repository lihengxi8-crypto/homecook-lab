先懂原理

每一步操作背后都有物理或化学机制。理解它，你才知道火大了/盐早了会发生什么。

🍳

再练技法

把炒煎炸烹、蒸炖焖烧归纳成"热 + 水 + 时间"的几套组合，技法不再零散。

🧩

最后创新

把食材、味型、技法当成可替换的变量，理解约束后再排列组合，做出属于你的菜。

Learning Map

学习全景图

五个篇章构成一条主线：从厨房里的科学，到下锅前后的每个决定。建议按顺序，也可随时跳读。

Roadmap

建议的学习路线

像搭知识树一样，先有根（原理），再长枝（技法），最后开花（创新）。

1

打地基：底层原理

搞懂热传递、美拉德反应、蛋白质变性、乳化与鲜味协同。这部分是后面一切的"物理引擎"。

热怎么传香从哪来为什么会嫩
2

下锅前：预处理

刀工决定受热均匀与入味；腌制上浆锁水嫩肉；焯水去腥定色断生。下锅前已经赢了一半。

刀工腌制上浆焯水
3

定味道：调味与酱料

认识每种调料的"职责"，再把味型理解成可计算的配比（兑碗汁）。从此调味不靠玄学。

五味平衡复合酱料味型公式
4

控火候：技法体系

把炒煎炸烹、蒸煮炖焖烧卤拆成"热源 + 介质 + 时间"，理解每种技法适合什么食材。

炒爆煎炸蒸炖烧勾芡时机
5

会收尾：实操 · 保存 · 创新

通用流程模板、翻车排查清单、剩菜保存的食品安全，最后用"变量化思维"做创新。

流程模板翻车排查冷藏复热创新组合
How to use

怎么用这个站

  • 看懂原理框：绿色「原理」框是机制解释，理科生可以深挖；只想做菜也能跳过。
  • 记住提示与翻车：黄色「贴士」和红色「翻车」是实操经验，最容易踩坑的地方。
  • 对照示意图：每个关键概念配 SVG 原理图，比纯文字更直观。
  • 用案例验证：每篇会配 1–2 道经典家常菜，把原理落到具体操作。

🧰 还有这些趁手工具

  • 互动工具：味型配比计算器、温度互动图、创新组合器（厨房工具）。
  • 查得到：按 / 全站搜索；正文术语自动加链接、悬浮即看释义（术语表）。
  • 对照学：案例菜库 与 食材百科（按部位/种类系统梳理）双向链接到原理与技法。
  • 选对器：厨具百科 梳理锅型与材质、刀型与钢材、微波炉/空气炸锅/烤箱等的加热原理。
  • 看演示：每篇底部「参考视频」精选检索词，一键直达 B站 / YouTube。
